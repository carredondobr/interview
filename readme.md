- Type: API Server
- Versioning: None

## Getting Started

#### Prerequisites
- Node version Hydrogen
- Docker

#### Initial Setup
1. Install dependencies
```
    npm i
```
2. Redis is required for this service
```
    npm run docker-compose:up
```
3. Start the service
```
    npm run start
```
or
```
    npm run watch
```

#### Testing
Run the following to run all of the tests
```
    npm run test
```
